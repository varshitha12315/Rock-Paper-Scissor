## Getting Started

### Option 1: Open directly (quickest)

1. Download/clone the project.
2. Open `DOM.html` in your browser.

### Option 2: Run via a local server (recommended)

Some browsers restrict certain behaviors when opening files directly. Running a local server avoids that and is best practice.

#### Using VS Code Live Server

1. Install the **Live Server** extension.
2. Right-click `DOM.html` â†’ **Open with Live Server**.

#### Using Python (if installed)

From the project folder, run:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000/DOM.html`.

## How to Play

1. Click **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects a move.
3. The message panel shows the result and the scoreboard updates.
